# Kaj je

Kolagen je protein ki daje koži strukturo in čvrstost. Elastin zagotavlja prožnost. Z leti se proizvajata manj - prezgodnja izguba je posledica UV sevanja, sladkorja, alkohola in cigaret.

# Kako ohraniti in zgraditi

## Prehransko

- odpraviti: predelani sladkor, alkohol, cigarete, kofein
- antioksidanti v prehrani (karotenidi, vitamin C in E)

## Topikalno - priporočeno

- Retin-A / retinol (dokazano najučinkovitejše)
- vlažilna krema z glicerinom in hialuronsko kislino
- sončna krema (prepreči razgradnjo)
- peptidi: Matrixyl, GHK-Cu

## Topikalno - optimalno

- vitamin C serum
- kemično luščenje
- mikroneedling
- rdeča svetlobna terapija
- laserji (IPL, CO2, Fraxel)

# Prioriteta

Retinol + sončna krema sta osnova. Brez sončne kreme je vse ostalo manj učinkovito.
