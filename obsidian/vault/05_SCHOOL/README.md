# 05 SCHOOL

Formalna šola — predmeti, naloge, izpiti, zapiski.

## Struktura
- `Predmeti/` — hub note za vsak predmet
- `Naloge/` — ena opomba na nalogo
- `Izpiti/` — priprave na preverjanja
- `Zapiski/` — zapisi s predavanj
- `Profesorji/` — kontakti in slogi poučevanja

## Tok dela
1. Predavanje → `Zapiski/Predmet - Datum`
2. Nov koncept → `Predmeti/Predmet/Koncept - Ime`
3. Naloga → `Naloge/Naloga - Predmet - Ime`
4. Pred izpitom → `Izpiti/Izpit - Predmet - Datum`
