# 02 AREAS

Stalna področja odgovornosti — stvari, ki nimajo končnega datuma, ampak jih vzdržuješ.

## Primeri področij
- Zdravje in telo
- Finance in denar
- Odnosi
- Osebni razvoj
- ARSS / kariera

## Razlika med AREA in PROJECT
- **Area** nima cilja — je stanje, ki ga vzdržuješ. (npr. *Zdravje*)
- **Project** ima jasen izid in rok. (npr. *Shujšati 5 kg do junija*)
