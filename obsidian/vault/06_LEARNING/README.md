# 06 LEARNING

Samostojno učenje izven formalne šole.

## Struktura
- `Teme/` — široka področja (npr. Programiranje, Investiranje)
- `Skill Trees/` — razčlenitev veščin
- `Study Plans/` — načrti učenja
- `Projects/` — praktična uporaba znanja
- `Summaries/` — povzetki virov

## Model
- Tema → Koncepti → Study Plan → Mini projekt → Refleksija
