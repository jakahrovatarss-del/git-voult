# 04 RESOURCES

Reference, članki, ideje, knjige, videi — vse, kar podpira tvoje učenje in projekte.

## Tipi virov
- Knjiga → `Vir - Knjiga - Naslov`
- Članek → `Vir - Članek - Naslov`
- Video → `Vir - Video - Naslov`
- Podcast → `Vir - Podcast - Naslov`
- Spletna stran → `Vir - Splet - Ime`

## Pravilo
Vsak vir naj ima:
- Zakaj je relevanten
- Ključne ideje (3–5 točk)
- Povezave na koncepte ali projekte, ki jih podpira
