## Kako Ustvariš Vir Note

Poimenovanje: `Vir - Naziv`

Struktura:
```

## Actionable insights

## Ocena (1–7)
```

AI prompt za to:
```
Povzemi ta [članek/video/knjigo] v Obsidian noto.
Format: Vir - [NAZIV]
Sekcije: Bistvo | Ključne ideje | Actionable | Povezave | Ocena
```
