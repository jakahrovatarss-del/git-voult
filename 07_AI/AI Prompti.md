# 📋 AI Prompti — Zbirka

Ready-made prompti za pogoste situacije v vaultu. Kopiraj, prilagodi, pošlji.

---

## 🏫 Šola

### Surov zapis → Strukturirana nota
```
Pretvori ta surov zapis v pregledno Obsidian noto.
Predmet: [[PREDMET]]
Dodaj frontmatter, izlušči glavne koncepte, poveži s sorodnima notama.
Sekcije: Kontekst | Glavne točke | Ključni koncepti | Povezave | Naslednji koraki
```

### Exam Review Sheet
```
Naredi kratek exam review sheet za [[PREDMET]] — tematika: [TEMA].
Format: bullet points z ključnimi definicijami in formulami.
Vključi 5 najpogostejših izpitnih vprašanj.
Poveži vsak koncept z obstoječo noto v vaultu.
```

### Flashcards
```
Iz te note ustvari 10 flashcard parov za [[Razmaknjeno Ponavljanje]].
Format: Vprašanje :: Odgovor
Vprašanja naj zahtevajo razumevanje, ne le zapomnitev.
```

### Samopreverjanje
```
Naredi 10 vprašanj za ustno preverjanje iz [[NOTA]].
3 lahka (definicije), 4 srednje (razumevanje), 3 težka (aplikacija).
```

## 📚 Solo Učenje

### Study Plan
```
Naredi [[Study Plan]] za temo: [TEMA].
Trajanje: [ČAS] dni/tednov.
Format: dan po dan, vsak dan max 1 koncept ali sklop.
Vključi: teorija → vaja → mini projekt → refleksija.
Poveži vsak sklop z obstoječimi notami ali predlagaj nove.
```

### Skill Tree
```
Ustvari [[Skill Tree]] za področje: [PODROČJE].
Struktura: osnove → srednje → napredno → master level.
Za vsako raven: kaj moraš znati, kateri vir je najboljši.
Označi, kaj že znam, kaj se učim, kaj je naslednje.
```

### Concept Breakdown
```
Razbij temo [[TEMA]] na učne module — od osnov do naprednega.
Za vsak modul: kaj je to, zakaj je pomembno, kako se nauči, mini projekt.
Kateri koncepti morajo biti znani prej (prerequisites)?
```

### Luknje v Znanju
```
Preglej moje zapiske za [[TEMA]].
Identificiraj luknje v razumevanju.
Predlagaj, katere note manjkajo in katere je treba razširiti.
Generiraj self-test po [[Feynmanova Tehnika|Feynmanovi tehniki]].
```

## 📰 Viri

### Povzetek Vira
```
Povzemi ta [članek/video/knjigo] v Obsidian noto.
Format: Vir - [NAZIV]
Sekcije: Bistvo | Ključne ideje | Actionable insights | Povezave | Ocena
Poveži z obstoječimi notami: [[KONCEPT1]], [[KONCEPT2]].
```

### Ideje iz Vira
```
Iz tega vira izlušči:
- 3 ideje, ki jih lahko takoj uporabim
- 2 koncepta, ki potrebujeta svojo noto
- 1 projekt, ki izhaja iz tega znanja
```

## 🔗 Linkanje in Struktura

### Najdi Manjkajoče Linke
```
Preglej to noto in predlagaj vse [[wikilinks]], ki manjkajo.
Posebej išči: osebe, koncepte, predmete, projekte, vire.
```

### Preveri Strukturo
```
Ali ta nota sledi pravilom iz [[01_SYSTEM/Vault Pravila]]?
Kaj manjka: frontmatter, sekcije, povezave, naslednji koraki?
```

### Poveži Dve Noti
```
Poišči vse konceptualne povezave med [[A]] in [[B]].
Predlagaj: ali je smiselna nova evergreen nota za skupni vzorec?
```
